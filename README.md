# homebrew-kubeexec

Homebrew tap for **kubeexec**.

kubeexec is a fast `kubectl exec` helper with fuzzy pod selection.

## Install

```bash
brew tap estebanmilaho/kubeexec
brew install kubeexec
```

## Docs

For usage, configuration, and examples, see the main repo:
